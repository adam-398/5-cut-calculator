## 5 Cut Method Calculator (React Native)

Simple calculator for squaring a cross-cut sled on a table saw using the William Ng 5 cut method.

## Formula
Adjustment = ((A - B) / 4) × (Pivot Distance / Cut Length)

Where A and B are the far and near side measurements of the 5th offcut strip.

## Built With
- React Native
- Expo
- TypeScript

## How to Run
Clone the repo, run `npm install` then `npx expo start`. Scan the QR code with Expo Go on your phone.
