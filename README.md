# 5 Cut Method Calculator
A woodworking utility app for squaring a cross-cut sled on a table saw using the William Ng 5 cut method — built with React Native and Expo.

## Features

- **5 cut method calculator** — input your measurements to get a precise fence adjustment in millimetres

## Formula

`Adjustment = ((A - B) / 4) × (Pivot Distance / Cut Length)`

Where A and B are the far and near side measurements of the 5th offcut strip. Result is in millimetres.

## Tech Stack

- TypeScript
- React Native
- Expo

## Getting Started

1. Clone the repository
2. Install dependencies
```bash
npm install
```
3. Start the development server
```bash
npx expo start
```
4. Scan the QR code with Expo Go on your phone
